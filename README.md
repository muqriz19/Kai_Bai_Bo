# Kai_Bai_Bo

**KAI BAI BO!** 

A Korea Rock Paper Scissor game programmed in HTML, CSS, JavaScript.

## Game Play

Its simple...

**Rock** beats **scissor** but dies from **paper**.
**Paper** beats **rock** but dies from **scissor**.
**Scissor** beats **paper** but dies from **rock**.

Understood? Good!
